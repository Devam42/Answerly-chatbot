How We Built DhruvAI: The Tech Behind the Talk

Step 1: Data Collection

The chatbot begins by collecting data from multiple sources to effectively address specific questions based on that information.

Step 2: Data Preprocessing

Using LangChain, Whisper, Self build youtube API, multiple tools and Google Generative AI embeddings, we transformed the data into vector data for easy search and retrieval.

Step 3: Data Storing

The vector data was securely stored in Pinecone, a cloud-based vector database built for fast and efficient data retrieval.

Step 4: Chatbot building

When a user sends a message, the chatbot first searches for an answer in Pinecone. If no answer is found, it analyzes the documents the user has attached and generates a response using Google Gemini. This process ensures the chatbot provides accurate replies based on the provided documents.
